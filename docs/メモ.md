## アプリ外ブラウザ

以下の処理で呼び出せる
端末にブラウザがないときに`ActivityNotFoundException`が発生するため例外処理はあったほうが良い

```kotlin
val intent = Intent(Intent.ACTION_VIEW, ＜ URL ＞)
context.startActivity(intent)
```
